Installing Eclipse

  We use Eclipse as our IDE. These are the commands to install it on Linux Ubuntu.

  • sudo apt install default-jre
  • sudo snap install --classic eclipse

We should also go over how to install the JavaFx which is a plug in that will allow you to build better gui.
